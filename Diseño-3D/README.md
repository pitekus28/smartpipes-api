# Diseño 3D — SmartPipes

## Descripción

Esta carpeta contiene los archivos de diseño 3D del proyecto SmartPipes. Aquí se incluye el modelo del encapsulado desarrollado en Fusion 360 y las exportaciones utilizadas para documentar y fabricar la carcasa del prototipo.

El diseño 3D fue desarrollado para alojar los componentes del sistema, proteger el hardware y permitir un montaje ordenado, considerando además acceso a elementos internos para mantenimiento y reparación.

---

## Contenido de la carpeta

### 1. Archivo fuente de Fusion 360

El archivo `.f3d` corresponde al modelo principal del encapsulado desarrollado en Fusion 360. Este archivo permite revisar el diseño paramétrico, editar dimensiones y visualizar el ensamble del sistema.

### 2. Archivo STL

El archivo `.stl` corresponde a la exportación del modelo 3D para fabricación o visualización de la carcasa.

### 3. Planos técnicos

Si se incluyen, los planos en PDF documentan las vistas principales, cotas y escala del encapsulado, permitiendo comunicar su geometría de forma técnica.

### 4. Renders o capturas

Si se incluyen, los renders muestran el encapsulado desde distintas vistas y ayudan a comunicar visualmente el diseño del sistema.

---

## Función del encapsulado

El encapsulado fue diseñado para:

* proteger los componentes electrónicos del sistema
* integrar físicamente sensores, microcontrolador y alimentación
* permitir el montaje de la carcasa mediante fijaciones mecánicas
* facilitar el acceso a componentes internos sin destruir el ensamble
* representar en 3D la solución física del proyecto

---

## Relación con el hardware

El diseño 3D no fue planteado solo como una carcasa estética, sino como parte funcional del sistema SmartPipes. Su propósito es contener y organizar los componentes del prototipo de forma coherente con el problema abordado, considerando un entorno asociado a ductos, humedad y necesidad de mantenimiento.

## Observaciones

Para la evaluación del proyecto, es importante que el diseño 3D represente el sistema de forma funcional y no solo estética. En caso de contar con un ensamble completo, el modelo debe reflejar cómo los componentes internos se integran físicamente dentro del encapsulado.
