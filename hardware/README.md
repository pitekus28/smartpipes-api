# Hardware
## Documentación de hardware

La documentación del hardware se encuentra en la carpeta `hardware/`.

Archivos principales:

- `hardware/BOM_SmartPipes.md`: lista de materiales del prototipo, cantidades, especificaciones, función de cada componente y costo aproximado.
- `hardware/esquematico_smartpipes_sensor1_uai.png`: esquemático de conexiones del prototipo SmartPipes Sensor 1 UAI.
- `hardware/README.md`: descripción general de la carpeta de hardware.

El esquemático muestra la conexión entre la batería LiPo 3.7V, el módulo Step-Up DC-DC 3.7V a 5V, la ESP32-S3 N16R8, el sensor ultrasónico SR04M-2 / AJ-SR04M, el sensor de humedad MH-RD Raindrops y la resistencia pull-down de 10kΩ.

Todos los GND del sistema comparten tierra común.
